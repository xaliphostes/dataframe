Dataframe Library API Reference

Core Components

Serie

Core

A column of data with elements of the same type, providing the fundamental building block of the library.

Dataframe

Core

A collection of named Series that can have different element types.

Functional Operations

forEach

Core

Iterates over each element in a Serie to perform an operation without changing the Serie.

map

Core

Transforms each element in a Serie using a callback function, returning a new Serie.

reduce

Core

Reduces a Serie to a single value by applying a function against an accumulator.

pipe

Core

Composes multiple operations into a single processing pipeline for Series.

filter

Core

Creates a new Serie containing only elements that satisfy a predicate function.

if_then_else

Core

Conditionally transforms elements in a Serie based on a predicate function.

switch_case

Core

Applies different transformations to elements based on multiple conditions.

Mathematical Operations

add

Math

Element-wise addition of two Series.

random

Math

Generates random number Series with various distributions.

bounds

Math

Finds the minimum and maximum values in a Serie.

sort

Math

Creates a sorted copy of a Serie in ascending or descending order.

Utility Functions

zip

Utility

Combines multiple Series into a single Serie of tuples.

unzip

Utility

Splits a Serie of tuples into multiple Series.

split

Utility

Divides a Serie into multiple equal-sized parts.

format

Utility

Converts a Serie to a formatted string representation.

Advanced Components

KDTree

Advanced

Spatial indexing structure for efficient nearest-neighbor queries.

Grid2D/Grid3D

Advanced

Structured grids for 2D and 3D data with associated attributes.

Mesh

Advanced

Triangulated surface mesh with vertex and triangle attributes.

Attributes Manager

Advanced

Manages and decomposes complex attributes in dataframes.
